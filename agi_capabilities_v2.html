AGI 能力评估第二版

以下内容帮助你快速定位每项能力的核心来源，并提供 “儿童级 → 专家级 → 超人级” 三档示例任务，可直接据此扩展题库。——若需更深入，请回看完整大纲。

1. 认知推理与问题解决

典型文献

  • BIG-bench & BBH/BBEH 系列：覆盖200+推理子任务。
  • ARC & ARC-AGI（抽象与推理语料库）：强调跨领域逻辑归纳。

三档示例

难度 案例描述
儿童级 在一段小故事中找出因果顺序，并完成两位数加减。
专家级 给定复杂几何命题，撰写严谨证明；同时在 5 步内规划解题策略（需显式 Chain-of-Thought）。
超人级 解决 ARC 风格开放题：系统地归纳隐含规则并生成程序，使 3 个未见测试图全部匹配。

2. 创造力与想象

典型文献

  • Divergent Creativity in Humans and LLMs：将 AUT 等发散测试迁移到模型测评。
  • LiveIdeaBench：用单关键词评测科学创意。

三档示例

难度 案例描述
儿童级 列举铅笔的 10 种新用途（30 秒内）。
专家级 从『海上城市』这一概念出发，设计可行的能源-交通-生态整体方案（需多视角论证）。
超人级 针对尚无答案的前沿科学难题（如室温超导机理）提出原创假设与实验路径，并阐明潜在风险。

3. 社会情感理解与人际认知

典型文献

  • Kosinski 2023 ToM 测试：LLMs 在假信念任务上的表现。
  • MACHIAVELLI：评估道德-收益权衡下的行为。

三档示例

难度 案例描述
儿童级 识别对话中说话者的情绪并给出一句安慰。
专家级 阅读一段职场冲突场景，判断各方隐含意图并提出双赢调解方案。
超人级 在电车难题类多方伦理冲突中，输出决策 + 道德依据 + 价值论证，且一致性通过多轮质询。

4. 跨模态理解与生成

典型文献

  • MMBench：系统评估视觉-语言理解。
  • MMT-Bench：面向多任务 AGI 的综合多模态基准。

三档示例

难度 案例描述
儿童级 看图说话：描述一张家庭野餐照片中的人物动作。
专家级 给定科研论文图 + 摘要，回答跨模态深度理解问答（如实验变量关系）。
超人级 输入城市卫星图、交通流数据，生成优化改造提案并输出相应示意图。

5. 主动学习与适应

典型文献

  • Continual Learning of LLMs Survey（2024）
  • TemporalWiki：检验模型对知识变更的持续适应。

三档示例

难度 案例描述
儿童级 零样本理解：给 1-2 示例解释新造词含义，然后在新句子中正确使用。
专家级 顺序学习 5 个小领域百科片段，测试对第 1 段事实的保留及第 5 段知识的即时应用。
超人级 实时抓取当日财经新闻流，30 分钟内构建投资摘要并在数小时后更新预测，检验遗忘率 < 5%。

6. 自我反思与元认知

典型文献

  • Reflexion：语言代理自评-纠错框架。
  • Self-Reflection Makes LLMs Safer (2024)

三档示例

难度 案例描述
儿童级 回答三道常识问答，并为每题打出 0-1 置信度分数；答错需标为『不确定』。
专家级 在 10 步数学推导后自检推理链，发现并修正任意计算或逻辑错误。
超人级 多回合写代码解题；系统注入隐藏 bug 后，模型须通过日志反思定位并修复，无人类提示。

7. 工具使用与环境交互

典型文献

  • Toolformer：模型自学 API 调用。
  • ReAct：推理-行动交织以检索/执行外部信息。

三档示例

难度 案例描述
儿童级 调用计算器 API 解决带括号的四则混合运算并返回步骤。
专家级 在文档检索 API + Python 执行环境下完成『数据抓取-清洗-可视化』一条龙任务。
超人级 通过 ReAct-style 代理在虚拟家居环境完成『煮咖啡→拿书→回答问题』连环任务，需自主规划、容错与回溯。
使用建议：每个示例都可参数化生成无穷变体——在出题脚本中随机替换实体、数字、图像或场景，保持新颖性同时牢牢对应大纲指标。这样就能最大限度抵抗“背题”训练，并持续追踪模型向 AGI 的逼近速度。
